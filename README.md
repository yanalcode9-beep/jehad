# 🚀 Dr. Jehad Professional Website

**Component-Based Architecture | Bilingual Support | Zero HTML Files**

## Features

✅ Object-Oriented Components
✅ English/Arabic Support (i18n)
✅ Server-Side Rendering
✅ RTL Support
✅ Modern ES6+ JavaScript
✅ Zero HTML files

## Quick Start
```bash
npm install
npm start
```

Open: http://localhost:3000

## Language Toggle

- English: http://localhost:3000/?lang=en
- Arabic: http://localhost:3000/?lang=ar

## Structure
```
src/
├── core/              # Base classes
├── components/        # UI components
├── config/           # Translations & data
├── styles/           # CSS-in-JS
├── scripts/          # Client-side JS
├── middleware/       # Express middleware
├── routes/           # Express routes
└── controllers/      # Request handlers
```
